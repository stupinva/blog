deb-пакеты с плагинами и темой для Wordpress

Тем же способом, который был описан в статье deb-пакет: тема arctic для dokuwiki собрал два deb-пакета с плагинами и один deb-пакет с темой для WordPress:

  1. wordpress-plugin-fckeditor-2.5.0_all.deb - редактор FCKEditor, без которого не будет работать следующий плагин,
  2. wordpress-plugin-multilingual-1.3.4.15_all.deb - локализация Wordpress для 14 языков, в том числе для русского,
  3. wordpress-theme-fluid-blue-1.7.1ru_all.deb - приятная "резиновая" тема.
